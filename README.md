
# Superstore Sales Analysis

## Project Overview
End-to-end data analytics project analyzing
e-commerce sales data to extract strategic
business insights for executive decision making.

## Tools Used
- Python (Pandas) - Data Cleaning
- MySQL - SQL Analysis
- Power BI - Dashboard

## Dataset
- 9800+ rows of e-commerce transactions
- 21 columns
- Variables: Sales, Region, Category,
  Customer, Ship Mode, Order Date and more

## Key Findings

1. Total revenue of $2.26M with stable
   4-day average shipping time

2. West region leads with $710K revenue
   vs South at only $389K

3. Phones and Chairs are top products
   generating $327K and $322K respectively

4. 47% of annual revenue concentrated
   in September to December holiday period

5. 2016 showed revenue dip despite higher
   orders suggesting heavy discounting.
   2017-2018 showed 30% explosive recovery

6. Sean Miller is top VIP customer
   spending $25,043 in only 5 orders

## Business Recommendations
1. Stock top products before September
   for holiday season surge
2. Run promotions in January and February
   to counter seasonal revenue drop
3. Increase marketing in West and East
   regions for maximum ROI
4. Investigate South region underperformance
